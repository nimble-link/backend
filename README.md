# backend

### Prerequisites
- [Docker](https://docs.docker.com/get-docker/)
- [go v1.14](https://golang.org/doc/go1.14)

### Development
- Install dependencies
```
./bin/setup_env.sh
```
- Run server

```
make run
```

Backend server will be served at: [localhost:8080](http://localhost:8080) by default
